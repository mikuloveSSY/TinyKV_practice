# TinyKV 环境配置记录

> 日期：2026-07-09

## 项目背景

- **课程**：华中科技大学计算机系统能力培养 - TinyKV 分布式键值存储系统
- **内容**：基于 Raft 共识算法，分 4 个 Project 逐步实现一个分布式 KV 数据库
  - P1: Standalone KV（单机键值服务）
  - P2: Raft KV（Raft 算法核心，选举/日志复制/快照）
  - P3: Multi-raft KV（多 Raft 组，成员变更/区域分裂/调度器）
  - P4: Transaction（MVCC + 快照隔离事务）
- **语言**：Go
- **代码仓库**：[github.com/tidb-incubator/tinykv](https://github.com/tidb-incubator/tinykv)

### 安装步骤回顾

```bash
# 1. 激活环境
conda activate tinykv_env

# 2. 安装 Go
conda install -c conda-forge go -y

# 3. 安装 C 编译器（Go 的 CGO 需要）
conda install -c conda-forge gcc_linux-64 -y

# 4. 安装 gopls 和 dlv（国内需换代理）
GOPROXY=https://goproxy.cn,direct go install golang.org/x/tools/gopls@latest
GOPROXY=https://goproxy.cn,direct go install github.com/go-delve/delve/cmd/dlv@latest

# 5. 将 ~/go/bin 加入 PATH（已写入 ~/.bashrc）
export PATH=$PATH:$HOME/go/bin
```
